# FEM Beam Lab (React)

Статическое React-приложение для расчета консольной балки (Beam FEM), готовое к публикации на GitHub Pages.

## Что уже сделано

- Полный Beam-расчет на TypeScript (матрица жесткости, ГУ, решение `K*u=f`, аналитика, сходимость).
- Улучшенная визуализация балки в SVG (контур балки, заделка, стрелка нагрузки, деформированная форма).
- Убран дублирующий график прогиба: оставлена одна основная визуализация + графики напряжений/сходимости.
- Solid-режим пока оставлен заглушкой для следующего этапа.

## Локальный запуск

```bash
npm install
npm run dev
```

Открыть: `http://localhost:5173`

## Production сборка

```bash
npm run build
npm run preview
```

## Деплой на GitHub Pages

1. Создай GitHub-репозиторий и запушь проект.
2. Убедись, что работаешь из папки `web`.
3. Выполни:

```bash
npm run deploy
```

Команда опубликует содержимое `dist` в ветку `gh-pages`.

4. В GitHub: `Settings -> Pages -> Build and deployment -> Source: Deploy from a branch`.
5. Выбери ветку `gh-pages` и папку `/ (root)`.

После этого сайт будет доступен по адресу GitHub Pages репозитория.
